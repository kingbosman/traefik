# Local Traefik HTTPS Proxy

A local Traefik setup that provides HTTPS for multiple projects using `.test` domains and trusted certificates via mkcert.

## Getting started

If you just cloned this repo, see [FIRST_SETUP.md](FIRST_SETUP.md) to get Traefik running on your machine.

## Adding a project

Once Traefik is running, see [ADDING_A_PROJECT.md](ADDING_A_PROJECT.md) to proxy a new local project through it.
