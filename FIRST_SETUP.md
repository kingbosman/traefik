# First-Time Setup

This only needs to be done once after cloning the repo.

## Prerequisites

- [Docker](https://docs.docker.com/get-docker/) with the Compose plugin
- [mkcert](https://github.com/FiloSottile/mkcert)
- `libnss3-tools` (required for Firefox/Chrome certificate trust)

On Ubuntu/Debian:
```bash
sudo apt install libnss3-tools
```

## 1. Install the local CA

```bash
mkcert -install
```

This makes your browser trust the certificates generated by mkcert.

## 2. Create the proxy Docker network

```bash
docker network create proxy
```

Traefik and all proxied projects share this network.

## 3. Generate the initial certificate

```bash
mkdir -p ~/path/to/traefik/certs
printf 'traefik.test\nlocalhost\n127.0.0.1\n' > ~/path/to/traefik/certs/domains.txt
cd ~/path/to/traefik && ./regen-cert.sh
```

To add domains later, edit `certs/domains.txt` and re-run `./regen-cert.sh` (see `ADDING_A_PROJECT.md`).

## 4. Add traefik.test to /etc/hosts

```
127.0.0.1 traefik.test
```

## 5. Start Traefik

```bash
cd ~/path/to/traefik && docker compose up -d
```

## 6. Verify

Open [https://traefik.test](https://traefik.test) — you should see the Traefik dashboard without any certificate warnings.
